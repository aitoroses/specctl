# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this community a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- being respectful and constructive
- giving actionable feedback
- assuming good intent while still being precise
- keeping discussions grounded in evidence
- helping improve the project for both human maintainers and agent operators

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- trolling or deliberately inflammatory behavior
- publishing private information without consent
- bad-faith security disclosures
- abusive treatment of contributors or maintainers

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This Code of Conduct applies within project spaces and public spaces when an individual is representing the project or its community.

## Reporting

Report unacceptable behavior to the maintainer through a private channel.
